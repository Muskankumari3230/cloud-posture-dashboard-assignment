# Cloud Posture Dashboard Assignment

**Candidate:** Muskan Kumari  

## Problem
Organizations use multiple cloud accounts, making it difficult to track and prioritize security misconfigurations.

## Solution
A simple 3-screen dashboard:
1. Dashboard overview with severity summary
2. Filterable misconfiguration list
3. Issue detail view with fix guidance and actions
